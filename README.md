# CBRoundedCorners

- 一行代码设置边框圆角


- 由于在swift语法里, 无法在extension中调用deinit方法, 所以图片的圆角处理只能通过子类化来实现.

To integrate CBRoundedCorners into your Xcode project using CocoaPods, specify it in your ```Podfile```:

```
pod 'CBRoundedCorners', '~> 1.0.2'
```
